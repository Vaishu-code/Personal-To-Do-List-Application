# Personal-To-Do-List-Application
A Python command-line based To-Do List application that allows users to manage tasks efficiently.
## Features
- Add tasks with title, description, and category
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using JSON file

## Requirements
- Python 3.x

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run: python todo.py

## File Structure
- todo.py → Main program
- tasks.json → Task storage
- README.md → Documentation
